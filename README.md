# 🎮 Tic-Tac-Toe - Retro Futuristic Edition

A sleek, futuristic Tic-Tac-Toe game built with vanilla HTML, CSS, and JavaScript. This project features a stunning Retro Futuristic aesthetic with neon colors, glowing effects, and smooth gameplay mechanics.

## ✨ Features

### 🎯 Game Modes
- **Player vs Player**: Classic head-to-head gameplay
- **Player vs Computer**: Challenge AI with adjustable difficulty
  - **Easy Mode**: Random moves for casual play
  - **Hard Mode**: Unbeatable AI using the Minimax algorithm

### 🎨 Visual Design
- **Retro Futuristic Theme**: Neon green (#00ff41) and pink (#ff0080) color scheme
- **Glowing Effects**: Text shadows, borders, and hover animations
- **Animated Background**: Subtle gradient animations for dynamic feel
- **Glassmorphism**: Modern backdrop blur and transparency effects
- **Orbitron Font**: Futuristic typography that enhances the sci-fi aesthetic

### ⏱️ Game Mechanics
- **Turn Timer**: 10-second countdown per turn with automatic timeout
- **Winning Highlights**: Visual feedback with pulsing animations
- **Symbol Selection**: Choose between X and O when playing against AI
- **Clean Interface**: Streamlined design without score tracking clutter

### 🎮 User Experience
- **Responsive Design**: Optimized for all screen sizes
- **Intuitive Navigation**: Easy-to-use menu system with back buttons
- **Smooth Animations**: CSS transitions, hover effects, and scale transforms
- **Game State Management**: Seamless game flow and restart options

## 🚀 How to Play

1. **Setup**: Open `index.html` in any modern web browser
2. **Choose Mode**: Select Player vs Player or Player vs Computer
3. **Select Difficulty**: Choose Easy or Hard AI (if applicable)
4. **Pick Symbol**: Choose X or O (in AI mode)
5. **Play**: Click cells to place your symbol
6. **Win**: Get three in a row (horizontally, vertically, or diagonally)
7. **Continue**: Use Rematch or Restart for new games
